## Vision

Our vision is **`Hope Open`**, which represents our desire for an open world. We believe that technology can break down barriers and promote the free flow of information.

## About

Hopen is a multi-platform proxy client based on [FlClash](https://github.com/chen08209/FlClash), with the technology stack migrated from Flutter to Rust.

Currently using **[GPUI](https://github.com/zed-industries/zed)** — a high-performance UI framework from Zed Industries.

## Features

✈️ Multi-platform: Windows, macOS and Linux

💻 Cross-platform desktop client with native performance

⚡ Built with Rust for memory safety and speed

🎨 Modern UI with Material You design inspiration

🔒 Open source and ad-free

## Technology Stack

- [GPUI](https://www.gpui.rs/) - High-performance UI framework
- Rust native

## Building

### Prerequisites

- Rust (latest stable)
- Git

### Clone the Repository

```bash
git clone https://github.com/sandhope/hopen.git
cd hopen
```

### Build Instructions

```bash
cargo build --release
```
