# 🕊 **Hopen**

Hope + Open = **Hopen**

/hoʊp/ + /ˈoʊpən/ = **/hoʊpən/**

<img src="resources/app-icon.png" alt="logo" width="300">

## 愿景

我们的愿景是 **`Hope Open`**，这代表了我们对开放世界的渴望。我们相信，技术可以打破壁垒，促进信息的自由流动。

## 关于

Hopen 是一个基于 [FlClash](https://github.com/chen08209/FlClash) 开发的多平台代理客户端，技术栈从 Flutter 迁移到 Rust。

当前使用 **[GPUI](https://github.com/zed-industries/zed)** — 来自 Zed Industries 的高性能 UI 框架。

## 特性

✈️ 多平台支持: Windows, macOS 和 Linux

💻 跨平台桌面客户端，原生性能

⚡ 使用 Rust 构建，内存安全且高效

🎨 现代 UI 设计，借鉴 Material You 设计语言

🔒 开源且无广告

## 技术栈

- [GPUI](https://www.gpui.rs/) - 高性能 UI 框架
- 纯 Rust 实现

## 构建

### 前置要求

- Rust (最新稳定版)
- Git

### 克隆仓库

```bash
git clone https://github.com/sandhope/hopen.git
cd hopen
```

### 构建说明

```bash
cargo build --release
```
